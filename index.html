Intelligent Parking System Using Machine Learning
Previous Next

This paper addresses the critical need for efficient parking management in the face of rapidly increasing automobile numbers. Our smart parking system leverages IoT and machine learning techniques to solve real-time parking challenges and uncertainties.

The system utilizes cloud technology and cameras to provide a comprehensive solution. A key feature is the development of a user-friendly graphical interface for both managers and end-users, ensuring smooth monitoring, management, and security of the parking system.

Key Features:
  • Real-time parking space status updates
  • Efficient management of limited and unreserved parking spaces
  • Automated parking slot reservation
  • Bengali number plate reading in real time
  • Automated billing and payment processing
  • Detection of incorrect and unauthorized parking
  • Integration with mobile applications for user convenience
  • Peak-hour traffic management
    • Technical Highlight:

    To automate the parking slot reservation process, we've used YOLOv5 model specifically trained to read Bengali number plates. This model achieves an impressive 97.4% accuracy in recognizing vehicle number plates. The model was trained on a dataset of 2,658 images of Bengali number plates, ensuring robust performance in real-world scenarios.

    Technologies Used:

    Python, YOLOv5, R-CNN, OpenCV

    This innovative approach significantly reduces the need for human intervention, saving time, money, and energy in parking management while providing highly accurate vehicle identification.
